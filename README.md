# Harihara Yashwanth Veldanda — Portfolio

A cinematic 3D scrollytelling portfolio built with Three.js. Scroll through a journey —
Hyderabad → SASTRA (Thanjavur) → Bengaluru → ASU → arrival — rendered as a procedural
isometric diorama, then into editorial sections covering experience, projects, and contact.

**Live:** _add your URL once hosted_

## Stack
- Vanilla HTML/CSS/JS — a single self-contained `index.html`, no build step
- [Three.js](https://threejs.org) r128 (local) for the 3D scene, with SSAO + tilt-shift depth-of-field post-processing
- [Lenis](https://github.com/studio-freight/lenis) for smooth scroll, [Syne](https://fonts.google.com/specimen/Syne) for display type

## Run locally
```bash
python -m http.server 4599
# then open http://localhost:4599
```
